# ArtisticGeek Studios — profile skeleton

A minimal landing-page foundation built around a responsive Three.js hero. The artwork combines fine animated wave lines, circuit traces, glowing current particles, and soft pointer-driven depth in the ArtisticGeek ivory-and-brush-gold visual language.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The project intentionally contains only the hero shell so future sections, routes, content, and CMS integration can be added without undoing a larger template.
